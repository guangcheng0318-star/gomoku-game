# 五子棋大對決 🎮

一個基於 GitHub Pages 的線上五子棋遊戲，支援本地雙人對戰和線上同步功能。

## 功能特色

- 🎯 **兩種遊戲模式**: 本地雙人 / 線上同步
- 📱 **響應式設計**: 支援電腦、平板、手機
- 🔗 **遊戲分享**: 透過連結分享遊戲狀態
- 🏆 **獨特規則**: 比連線數量而非單一連五
- 🎨 **美觀界面**: 現代化 UI 設計

## 遊戲規則

1. 兩位玩家輪流下棋（○ 和 ✕）
2. 下滿整個棋盤後計算分數
3. 連成 5 子算 1 條連線
4. 水平、垂直、對角線連線都計算
5. 連線數量多者獲勝

## 部署到 GitHub Pages

1. Fork 這個倉庫
2. 在倉庫設定中開啟 GitHub Pages
3. 選擇 `main` 分支作為來源
4. 訪問 `https://你的用戶名.github.io/倉庫名稱`
